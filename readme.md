\# Projekt Informatyka II - System SCADA (Python + PyQt5)



\## Opis Projektu

Projekt stanowi aplikację typu SCADA (Supervisory Control and Data Acquisition) wizualizującą proces przemysłowy mieszania substancji. Aplikacja została zrealizowana w języku Python przy użyciu biblioteki PyQt5.



\## Scenariusz Procesu

Symulacja przedstawia automatyczny proces mieszania dwóch substratów:

1. Zlewanie: Odpowiednia ilość substratu z obu zbiorników jest spuszczana do zbiornika mieszalnika.

2\. Mieszanie: Po napełnieniu, uruchamia się mieszadło.

3\. Opróżnianie: Gotowa mieszanina jest grawitacyjnie spuszczana do zbiornika produktu końcowego.



\## Funkcjonalności

* Wizualizacja graficzna: 4 zbiorniki, system rur oraz animowane mieszadło.
* Dynamika: Płynna animacja przepływu cieczy i śmigła mieszadła.
* Automatyka: Logika stanowa sterująca kolejnymi etapami procesu.
* Konfiguracja: Możliwość zadania parametrów początkowych (ilość substratów) przed startem symulacji.
